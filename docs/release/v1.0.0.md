# 🚀 Release Notes v1.0.0 – FLET Monitoring System

## 🆕 New Services

### ⏰ Real-time Service

- Real-time sensor data collection and processing
- WebSocket-based real-time data streaming
- Sensor status monitoring

### 🚥 Thresholds Service

- Sensor threshold configuration and management
- Threshold breach alerting
- Dynamic threshold adjustment

### 🚩 Location Service

- Sensor location information management
- Geographic data handling
- Location-based sensor grouping

### 📈 Aggregation Service

- **Extended data metrics**:
  - Apparent temperature: `pcv_temperature_max`, `pcv_temperature_avg`, `pcv_temperature_min`
  - Temperature: `temperature_max`, `temperature_avg`, `temperature_min`
  - Humidity: `humidity_max`, `humidity_avg`, `humidity_min`

- **Time-based aggregation**:
  - Year (`yyyy`): monthly aggregation
  - Month (`yyyyMM`): monthly if span ≥ 4 months, otherwise daily
  - Day (`yyyyMMdd`): hourly aggregation

- **API response**: single `location` replaced with `locations` list
