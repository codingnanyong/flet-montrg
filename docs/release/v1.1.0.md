# 🚀 Release Notes v1.1.0 – FLET Monitoring System

## 🆕 New services

### 📚 Integrated Swagger Service

- Single Swagger UI for all backend APIs
- API proxy: single entry point for all services
- Port: **30001**

### 🌐 Web Service (Frontend)

- Web UI for API gateway and monitoring
- Svelte 4 + Vite 5
- Port: **30000** (use `?apiBase=http://<host>:30001` to point to integrated Swagger / proxy)

### 🔔 Alert services

- **Alert Service** (30006): alert creation and management
- **Alert Subscription Service** (30007): subscription by location/sensor/threshold
- **Alert Notification Service** (30008): notification delivery per subscriber
- Hierarchical matching: factory → building → floor → area

### 🔗 Sensor-Threshold Mapping Service

- Map thresholds to sensors
- Valid-from/valid-to and activation control
- Port: **30009**

## 📌 Changes and additions

### 📁 Project structure

- **config/** added at repo root for shared configuration
